# ShaderLearning

Shader 的学习项目, 用于学习 Shader.

## 项目结构

- [Assets/Prefabs](Assets/Prefabs) 一些随便摆的场景来验证某些反射/折射相关的 Shader
- [Assets/Scenes](Assets/Scenes) 各个场景, 每个场景是某个主题下的一系列 Shader 的展示
  - [标准Shader](Assets/Scenes/StandardShader/) 构建一个标准的~~最简单的~~光照模型
